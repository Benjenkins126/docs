---
title: "Authentication"
description: "Learn how to authenticate your requests"
---


1. Go to https://www.pacekb.com/app/admin/sign-in and sign in with your credentials
2. From the sidebar select "Settings"
3. You should now select "API Keys" from the Settings navigation
4. Click "+ Add New" and copy the API Key generated.

<Warning>This API Key will never be re-shown. Ensure you copy it to a safe place</Warning>

5. This API key can be used in all requests to the PaceKB API by placing it in an Authorization header.

```bash
'Authorization': 'Bearer <api_key>'
```

### Permissions

Knowledge Base administrators can restrict who can generate API keys via the User Roles page. If you cannot access the API Keys page you'll need to contact your site administrator.
